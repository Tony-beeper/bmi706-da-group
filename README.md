## To Run locally:

1. Download the required dependencies from `requirements.txt` using the following command:

    ```bash
    pip install -r req
    ```
replace [req] with requirements

2. Then, run the app in ```./``` using the code below:

    ```bash
    streamlit run streamlit_app.py
    ```
